About the train

The Class 756 is a FLIRT tri-mode multiple unit used on Regional services.

Regional

Service badges are now colour-coded to match the service type style, so Intercity, Regional, Semi-fast, and Shuttle stand out more clearly.

Database notes

  • It is bundled with the Class 231 and Class 755.
  • It can multiple-work within class and with Class 231 and Class 755.
  • It uses AC, battery, and diesel traction modes.
  • One livery is listed on the official wiki.
