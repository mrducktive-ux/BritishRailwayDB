About the train

The Class 220 Voyager is a four-car diesel-electric multiple unit used on Intercity services. It comes as part of the Voyager Pack.

This page is part of the unofficial British Railway DB project and links back to the official wiki for further information.

Database notes

  • It is bundled with the Class 221.
  • It reaches 125 mph.
  • It can multiple-work within class and with the Class 221.
  • Three liveries are listed on the official wiki.
