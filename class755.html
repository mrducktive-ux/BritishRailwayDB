About the train

The Class 755 is a FLIRT bi-mode multiple unit operated on Regional and Semi-fast services.

RegionalSemi-fast

Service badges are now colour-coded to match the service type style, so Intercity, Regional, Semi-fast, and Shuttle stand out more clearly.

Database notes

  • It is bundled with the Class 231 and Class 756.
  • It can multiple-work within class and with Class 231 and Class 756.
  • It uses 25 kV AC overhead power as well as diesel power.
  • One livery is listed on the official wiki.
