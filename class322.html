About the train

The Class 322 is a four-car electric multiple unit operated on Regional and Semi-fast services.

This page is part of the unofficial British Railway DB project and links back to the official wiki for further information.

Database notes

  • The Class 321 is required before this train can be purchased.
  • It can multiple-work within class and with the Class 321.
  • One livery is listed.
  • It runs on 25 kV AC overhead power.
