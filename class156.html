About the train

The Class 156 Super Sprinter is a two-car diesel-hydraulic multiple unit operated on Regional services.

This page is part of the unofficial British Railway DB project and links back to the official wiki for further information.

Database notes

  • It can work in multiple with several Sprinter classes and the Class 170.
  • Eight normal liveries and two special liveries are listed.
  • It reaches 75 mph.
  • It was available from Version 1.0.0.
