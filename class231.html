About the train

The Class 231 FLIRT is a four-car diesel-electric multiple unit operated on Regional services.

This page is part of the unofficial British Railway DB project and links back to the official wiki for further information.

Database notes

  • The Class 755 is required before this train can be purchased.
  • It is bundled with the Class 755 and Class 756.
  • It can multiple-work with the Class 755 and Class 756.
  • One livery is listed.
