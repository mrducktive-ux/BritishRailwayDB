About the train

The Class 159 is a three-car diesel multiple unit used on Regional and Semi-fast services. It is based on the Sprinter family and is purchased as an upgrade from the Class 158.

This page is part of the unofficial British Railway DB project and links back to the official wiki for further information.

Database notes

  • The Class 158 is required before the Class 159 can be purchased.
  • It can multiple-work with Class 142, 150, 153, 156, 158 and 170.
  • It reaches 90 mph.
  • Two liveries are listed on the official wiki.
