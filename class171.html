About the train

The Class 171 is closely related to the Class 170, but uses a Dellner coupler. It is available in two, three, and four-car formations.

Players must own the Class 170 before purchasing the Class 171.

Database notes

  • Operated by Regional.
  • Supports multiple working within its own class.
  • Contains one livery.
  • Uses diesel multiple-unit traction.
