About the train

The Class 221 Super Voyager is a five-car diesel-electric multiple unit operated by Intercity. It is included in the same Voyager Pack as the Class 220.

This page is part of the unofficial British Railway DB project and links back to the official wiki for further information.

Database notes

  • It is bundled with the Class 220.
  • It reaches 125 mph.
  • It can multiple-work with the Class 220.
  • Four normal liveries and two special liveries are listed.
